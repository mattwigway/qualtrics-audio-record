# Qualtrics Audio Recorder

This adds an audio recorder to a Qualtrics survey.

## How to use

- Add a file input where you want the audio recorder
- Add the contents of `qualtrics-audio-record.js` to the Javascript on that question
- Add the contents of `qualtrics-audio-record.css` to the Custom CSS under look and feel

Note that this relies on undocumented Qualtrics internal features of the file upload box, so may stop working at some point. If anyone knows a documented way to upload files let me know!
